# Pinned build/runtime closure for the macOS universal2 builds.
#
# Why this file exists: beets hard-declares numba + scipy (which pull llvmlite),
# but its code never imports them and we exclude them from the Nuitka freeze.
# On x86_64 macOS there are NO numba/llvmlite wheels, so a normal install tries
# to build llvmlite from source (needs LLVM) and fails. Installing this list
# with `--no-deps` sidesteps that: numba/scipy/llvmlite are simply never
# resolved, and both macOS arches get an identical module set (clean lipo merge).
#
# Maintenance: this is the full transitive closure MINUS {numba, scipy, llvmlite},
# resolved for Python 3.12 (the CI build version). Regenerate when bumping beets
# or the Python build version. The Python 3.13-only stdlib backports
# (audioop-lts, standard-aifc/chunk/sunau) are intentionally absent — on 3.12
# those modules are still in the standard library.
